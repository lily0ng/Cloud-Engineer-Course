
**By Data Model**

1. **Relational Databases (SQL)**
   - MySQL
   - PostgreSQL
   - Oracle Database
   - Microsoft SQL Server
   - SQLite

2. **NoSQL Databases**
   - **Document Databases:** MongoDB, CouchDB
   - **Key-Value Stores:** Redis, DynamoDB
   - **Column-Family Stores:** Cassandra, HBase
   - **Graph Databases:** Neo4j, Amazon Neptune

**By Deployment/Service Model**

1. **On-Premises Databases**
2. **Cloud Databases**
   - Managed DB Services (AWS RDS, Azure SQL Database, Google Cloud SQL)
   - Database-as-a-Service (DBaaS)

**By Purpose/Specialization**

1. **Operational Databases (OLTP - Online Transaction Processing)**
2. **Analytical Databases (OLAP - Online Analytical Processing)**
   - Data Warehouses: Amazon Redshift, Google BigQuery, Snowflake
3. **Time-Series Databases:** InfluxDB, TimescaleDB
4. **In-Memory Databases:** Redis, Memcached
5. **Search Engines:** Elasticsearch, Solr
6. **Ledger Databases:** Amazon QLDB

**New Trends**

- **Multi-Model Databases:** support multiple data models
- **Serverless Databases:** automatic scaling
- **Edge Databases:** for IoT and edge computing
